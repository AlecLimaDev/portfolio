MINHAS SKILLS

HTML

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS

a função do CSS é justamente otimizar o aspecto visual das páginas, de uma maneira mais organizada e menos complexa na hora realizar manutenção.

JavaScript

O JavaScript é uma linguagem de programação popular e de alto nível que permite criar conteúdo dinâmico em páginas web e é executada nos navegadores dos usuários finais. É uma linguagem interpretada e orientada a objetos com uma ampla gama de recursos para desenvolvimento de aplicativos web.

React

O React é uma biblioteca de JavaScript desenvolvida pelo Facebook que permite criar interfaces de usuário interativas e reutilizáveis. Ele usa uma abordagem baseada em componentes para simplificar o desenvolvimento de aplicativos web complexos.

Next.js

O Next.js é uma estrutura de desenvolvimento web em React, que oferece recursos como renderização do lado do servidor, roteamento e pré-renderização, tornando o desenvolvimento de aplicativos web mais eficiente e escalável. Ele é flexível o suficiente para ser usado em qualquer projeto React.

Git

Git é o sistema de controle de versão open source mais usado no mundo atualmente! Ele é usado para controlar o histórico de alterações de arquivos e principalmente de projetos de desenvolvimento de software. Ele permite mais flexibilidade no fluxo de trabalho, segurança e desempenho.

Node

Node.js é um ambiente de execução de JavaScript do lado do servidor, que utiliza o motor V8 do Google Chrome. Ele permite o desenvolvimento de aplicativos escaláveis e de alta performance usando a linguagem JavaScript tanto no frontend quanto no backend, simplificando o processo de desenvolvimento de aplicativos web.

TypeScript

O TypeScript é uma linguagem de programação open-source desenvolvida pela Microsoft que é uma extensão do JavaScript, adicionando tipagem estática e outros recursos avançados para tornar o desenvolvimento de aplicativos mais seguro e produtivo.

Styled-Components

O Styled Components é uma biblioteca JavaScript para React que permite escrever CSS dentro do código, criando componentes com estilos únicos e encapsulados, tornando a criação de interfaces de usuário mais eficiente.

Firebase

O Firebase é uma plataforma de desenvolvimento de aplicativos móveis e web, desenvolvida pelo Google, que oferece uma variedade de recursos como autenticação, armazenamento de dados, hospedagem de aplicativos, entre outros, simplificando o processo de desenvolvimento e lançamento de aplicativos.
